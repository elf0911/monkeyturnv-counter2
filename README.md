# monkeyturnv-counter2

モンキーターンV専用カウンター / 設定判別アプリです。

## 管理ファイル

- index.html
- app.js
- style.css
- manifest.json
- service-worker.js
- README.md

## 今回の反映内容

- iPhone向けの既存カードUIを維持
- 設定判別画面を既存フォーマットに合わせて調整
- 初期チェックは「総ゲーム数」「AT初当たり」のみON
- 5枚役は総ゲーム数 / 通常ゲーム数の分母切替に対応
- 確定・濃厚は同義として期待度へ反映
- 通常判別では最低期待度1%を維持
- 影響度を星表示
- AT直撃は契機別に重みを変更

## 参照

設定確定・濃厚系は、ななプレス「スマスロモンキーターンⅤ 設定差・設定示唆演出まとめ」を参照。
