# モンキーターンV カウンター

GitHub Pages用のスマホ向けカウンターです。

## stage1-ui

- 6ファイル構成を維持
- iPhone向けカードUIを維持
- AT直撃を5契機に分割
- チャージ後セリフを短縮表示
- AT中示唆を追加
- エンディング中サブ液晶を追加
- SGメダル/トロフィー/舟券を実機仕様に整理
- 設定判別画面から総ゲーム数を除外


## stage1-ui-order-layout

- カウンターの並び順を入力頻度ベースに変更
- 激走チャージ中を「激走チャージ中アイテム」に変更
- 設定判別画面の並びもカウンター順に統一
- エンディング中サブ液晶などを3列レイアウトに変更
- 回数列を固定幅・右寄せにして「回」が縦に揃うように調整
- 偶数設定示唆/濃厚の表記を「偶数示唆」「偶数濃厚」に統一

## stage1-ui-order-layout-v2

- カウンター画面は変更せず維持
- 設定判別の展開内を3列レイアウトに修正
- 左列・右列は左寄せ、中央の回数列だけ右寄せ
- 2桁/3桁でも「回」の位置が揃うように調整


## stage1-ui-judge-layout-v12

- 公表値表を左列固定・右側横スクロールに調整
- 表の余白を圧縮
- AT初当たり/5枚役/即優出の2行表の見た目を統一
- SGメダル公表値を黒メダル/黒メダル後に整理
- トロフィー公表値を通常/黒メダル後に整理
- 設定画面にレア小役の説明を追加
